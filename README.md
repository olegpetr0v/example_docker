# example_docker

Цель данного проекта продемонстрировать пример развертывания набора сервисов посредством Docker.

Из регистра DockerHub загружаются образы из которых собираются базовые образы, используемые для конкретных сервисов.
Инструкции по сборке базовых образов располагаются в ./dockerfiles
Используются следующие базовые образы

Из базовых образов запускаются контейнеры.
Инструкции для запуска контейнеров находятся в ./service

В директории ./volumes располагаются директории, которые монтируются в соответсвующие контейнеры.

Управление сборкой контейнеров осуществялется с помощью docker-compose.yml
Управление контейнерами осуществляется с помощью Portainer, который также запускается в контейнере.
